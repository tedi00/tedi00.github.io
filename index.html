Basics of C

Basic declarations and expressions

Declarations

These are the main data types for C.

To cut things short though, you've got char, int, float and double. If you want you can read up on short/long or signed/unsigned.

Here are their format specifiers.

Variable type char int float double
Format specifier %c %d %f %lf

This will display a value like 35.2222 as 35.22

Here is the syntax

Declaration only variable_type name;

Giving a value name = value;

Declaration and value variable_type name = value;

Remember that the variable name will always be on the left, and the value given to it will be on the right side.
It's not quite like maths where both sides of the = are interchangeable.

I/O

printf and scanf are the input and output expressions for C when dealing with console applications.

Here is the syntax

For printf - printf("text and/or format_specifier", variable_of_format_specifier);

For scanf - scanf("format_specifier", &variable_of_format_specifier);

Keep in mind you can round up a floating point decimal in C as such - %.2f

When outputting text, you can use things such as /n, /b, /r and /t.

/n - newline, similar to enter

/b - backspace, moves back one character without deleting anything

/r - carriage return, moves back to the start of the line and replaces characters from there

/t - tab, spaces similarly to pressing tab

I suggest you try these yourself to see how they work in practice.

Conditional statements

If/else

A conditional statement runs code only if the condition is met - if or if/else statements.

This technically means they only accept booleans - true or false.
If the condition is met, we get true, and the code runs through if. If the condition is not met, we get false, and the code runs through else, if we have one.

Here are the operators used in conditional statements

Operator == != < > <= >=
Meaning Equals Not equals Smaller than Bigger than Smaller or equals Bigger or equals

And here is the syntax for a normal if/else.

if(a)

{ run code for a == true }

else

{ run code for a == false }

Remember that = is used to give value while == is used for comparisons.

Booleans

Now, an if/else statement can contain a whole expression inside its brackets, like if(x==0) for example. It can however also contain one single variable, known as a boolean.

Both the expression and the boolean can only have set values, namely true or false.

A boolean is a special type of variable, declared with _Bool (case sensitive).

If a value is given to it, it can only be _Bool a = true; or _Bool b = false; in the same way that x==0 can only be true or false.

Therefore if we look at the syntax for the if/else conditional statement, the a in the brackets can be either an expression or a single boolean.

Now let's take a look at this line of code.

if(a)

{int newval = 1;}

else

{newval = 0;}

This code is not correct. There's a declaration for a variable called newval.

If you need to declare a variable inside an if statement, it's always good practice to also declare it in the else statement. Usually these values will be used after the if/else, and if the value is only declared in the if but the code only runs through else, that value is technically never declared.

Conditional Operators

Well, these are practically similar to if/else statements except they don't cover a range of uses as broad.
They're mainly used to return a single value based on a condition. Like this.

(condition? value_if_true : value_if_false)

Because they return a single value they're good at assigning a value to a variable based on conditions.
Let's say I want my x value to be 10 or 20 based on if y is even or odd.

int x = (y%2==0? 10 : 20);

So if y/2 has a remainder of 0 (y%2==0) then x becomes 10, but if the remainder isn't 0 (the number is odd and the remainder is 1) x becomes 20.

Switch statements

A switch - also called a switch/case statement, works by taking a variable and doing different things based on the value of that variable.
Kind of like an if/else, but with several more specific options instead of a range.

Here's an example of how it works.

switch(variable)

{

case value1:

run code for variable == value1

break;

case value2:

run code for variable == value2

break;

default:

run code for when variable has a value we didn't predict

}

You can use as many cases as necessary, and breaks as well as the default case is optional.

break; works in switch statements, loops etc. as a way to stop and move past it.

If we have a break in the first case, the program doesn't check through every other case below it and moves past the switch entirely, making it slightly faster.

default is a special type of case, containing every case we didn't predict. Think of it as error handling in a way, or a safety net.

Loops

A loop executes the sequence of statements inside it repeatedly until the stated condition becomes false.

It usually consists of the body of the loop - where the code repeats - and a control statement which usually controls the conditions.

We're gonna look at all three loops in C - for, while and do while.

For loop

The for loop has a control statement divided in 3 different parts, and then a body where the code runs.

The control statement has an initial value, a condition to be met for the body to run, and an incrementation or decrementation of the value from the initial.

Let's take a look at the syntax.

for(initial_value; condition; incrementation_or_decrementation)

{ statements to be repeated inside body }

It's a good idea to declare the initial value before the for loop, and give it a value in the control statement.

While loop

A while loop is the simplest form of loop one can do. Less complex than for and slightly less efficient as well.

Like the for loop, it's an entry-controlled loop, meaning the condition is checked before the code on the body is ran.
If the condition starts off false, the code is never executed.

Here is the syntax.

while(condition)

{ statements to be repeated inside body }

Let's also a practical example of a while condition.

This example takes for granted that we've declared a previous int num=0; before the while loop, and displays numbers from 0 to 9.

As you can see, there's an incrementation in the form of num++; in while as well.

However this one is done manually inside the body of the while instead of the control statement.
If it's not added, the while loop continues forever.

Do While loop

This one is like the while loop, but it is not an entry-controlled loop.

An exit-controlled loop like this will let the code in the body run once, then check if the condition is met.
If it is, the code runs again in a loop until the condition is false.

Let's look at the syntax.

do {

statement runs once then condition is checked to run again

} while (condition);

And here is example.

int a=1;

do {

printf("%d ", a);

a++;

} while(a<0);

In this code, a is never smaller than 0, yet the printf is run once anyway, before the condition is checked. The output is 1.

Other than this one difference - the fact that the code runs once before checking the condition, while and do while are practically the same.
